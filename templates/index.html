Quip Wallet Explorer

About this Explorer

This page queries the Quip Testnet via a public bootnode RPC endpoint. To keep responses fast, only the last 50 blocks are scanned for events. The wallet balance (free, reserved, total) is fetched live and is always current.

Event history is limited to recent activity. For a complete historical scan (all blocks from genesis), run the wallet locally against your own validator — scanning the full chain over a public RPC would take several hours.

The ranking shows all wallets that have been searched on this page, sorted by total QUIP balance.
